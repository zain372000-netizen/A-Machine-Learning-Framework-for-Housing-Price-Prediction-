# Jordan Housing Price Dataset

## Overview

This repository contains the dataset used in the research paper:

**Investigating the Impact of Spatial and Economic Heterogeneity on Housing Price Prediction in Jordan Using Machine Learning**

The dataset consists of apartment listings collected from real estate platforms across Jordan. It combines property characteristics with regional demographic and economic indicators to support research on housing price prediction using machine learning.

---

## Dataset Information

- **Number of observations:** 2,989 apartments
- **Number of features:** 15
- **Target variable:** Price_JOD
- **File format:** Excel (.xlsx)

---

## Dataset Variables

| Variable | Description |
|-----------|-------------|
| Price_JOD | Apartment asking price in Jordanian Dinars (JOD). This is the prediction target. |
| Area_m2 | Apartment area in square meters. |
| Bedrooms | Number of bedrooms. |
| Bathrooms | Number of bathrooms. |
| Furnished | Furnishing status (e.g., Furnished or Unfurnished). |
| Parking | Availability of parking space. |
| Elevator | Availability of an elevator in the building. |
| Balcony | Availability of a balcony. |
| Garden | Availability of a private or shared garden. |
| Governorate | Governorate where the apartment is located. |
| Residential_Area | Residential district or neighborhood. |
| Population | Population of the corresponding residential area. |
| Population_Density | Population density of the residential area. |
| Unemployment_Rate | Estimated unemployment rate for the area. |
| Economic_Development | Economic development indicator associated with the residential area. |

---

## Dataset Characteristics

The dataset integrates two categories of information:

### Property Features

- Apartment area
- Number of bedrooms
- Number of bathrooms
- Furnishing status
- Parking availability
- Elevator availability
- Balcony availability
- Garden availability

### Spatial and Socioeconomic Features

- Governorate
- Residential area
- Population
- Population density
- Unemployment rate
- Economic development indicator

These variables enable the investigation of how local socioeconomic conditions influence housing prices in addition to traditional property characteristics.
